nano
========
Lightweight http parser implementation. nanohttp is a fork from mensley's tinyhttp but it has a working example, a Makefile and the source code minified withing two files.

* `<ctype.h> - tolower`
* `<string.h> - memcpy`
* `<string.h> - strncmp`

Compiling
---------
$ make
$ ./example
